# kudosu

Simple sudoku game made with React.

# Features

- [x] Sudoku board
- [x] Solution check
- [x] Editable number tiles
- [x] Blocked puzzle-preset tiles
- [ ] Marked tiles
- [ ] Different boards
- [x] Language setting, of course
- [ ] Anything else that is missing
